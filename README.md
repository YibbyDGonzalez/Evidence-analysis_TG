# NLP Evidence Analysis

## Descripción

Este proyecto realiza un análisis de texto sobre literatura científica en medicina basada en la evidencia utilizando técnicas de NLP.

## Contenido

En el archivo **Descriptiva_Corpus.ipynb** se encuentra:

* Análisis lingüístico
* Gráficas descriptivas
* Análisis de bigramas y trigramas
* Análisis exploratorio de datos (EDA) con embeddings

## Corpus utilizados

* *Users' Guides to the Medical Literature: A Manual for Evidence-Based Clinical Practice*
* *Painless Evidence-Based Medicine*

## Herramientas

* Python
* NLTK
* PyMuPDF
* Técnicas de procesamiento de lenguaje natural (NLP)

## Notas

El detalle completo del proceso, incluyendo limpieza, tokenización y modelado, se encuentra documentado dentro del notebook.
